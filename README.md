# Capstone Project 2025
## Football Player Valuation Prediction

### Project Structure
- `src/` : All Python modules for data processing and modeling
- `data/` : Data directory structure (place raw data here)
- `models/` : Trained models (generated locally)
- `notebooks/` : Jupyter notebooks for exploration

### Setup
1. Clone: `git clone git@github.com:Samuelvermeulen/capstone-project-2025.git`
2. Install: `conda env create -f environment.yml`
3. Run: `python main.py`

### Note
Data files are excluded via .gitignore. Place raw data in `data/raw/` and run `src/clean_final.py`.
